stPagination
filter in module stPagination

Description

Initialized the pagination and returns a new limited sub-array with an offset controlled by a stPagination directive. The initialized pagination object handles the filtering, correct calculation of offsets and pages.

   <ul>
     <li ng-repeat="user in users | stPagination">{{ user.name }}</li>
   </ul>

Usage with other filters

If you want to apply other filters to you collection before the pagination filter, you have to pass the originalCollection to the filter. Otherwise the pagination object can not be attached correctly.

   <ul>
     <li ng-repeat="user in users | filter:{name: usernameFilter} | stPagination:users">
       {{ user.name }}
     </li>
   </ul>

Usage

In HTML Template Binding

{{ stPagination_expression | stPagination[:originalCollection] }}

In JavaScript

$filter('stPagination')(inputCollection[, originalCollection])

Parameters

ParamTypeDetails
inputCollectionArray

originalCollection

Array
