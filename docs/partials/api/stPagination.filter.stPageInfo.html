stPageInfo
filter in module stPagination

Description

Returns information about pagination properties. Array must be initialized with the pagination filter.

   {{ users | stPageInfo:'total' }}
   {{ users | stPageInfo:'currentPage' }}
   {{ users | stPageInfo:'totalPages' }}
   {{ users | stPageInfo:'startIndex' }}
   {{ users | stPageInfo:'stopIndex' }}

Usage

In HTML Template Binding

{{ stPageInfo_expression | stPageInfo:propertyName }}

In JavaScript

$filter('stPageInfo')(collection, propertyName)

Parameters

ParamTypeDetails
collectionArray

Array that was initialized by the stPagination filter

propertyNameString

Name of the pagination property

  • 'total' total number of elements in the collection
  • 'currentPage' index of the current page
  • 'totalPages' total number of pages
  • 'startIndex' index of the first element on the current page
  • 'stopIndex' index of the last element on the current page

Returns

Number

The value of the specified pagination property.
